# 游戏机制详解

## 🎮 核心玩法机制

### 1. 钓鱼流程
```
点击屏幕 → 鱼钩下放 → 碰撞检测 → 捕获鱼 → 自动回收 → 获得分数
```

### 2. 鱼类系统

#### 6 种鱼类
| 鱼类 | 分值 | 稀有度 | 速度 | 出现深度 | 基础概率 |
|------|------|--------|------|----------|----------|
| 小鲤鱼 | 10 | ⭐ | 50 | 100-300 | 40% |
| 鲫鱼 | 20 | ⭐⭐ | 60 | 200-400 | 30% |
| 草鱼 | 30 | ⭐⭐ | 70 | 300-500 | 15% |
| 鲤鱼王 | 50 | ⭐⭐⭐ | 80 | 400-600 | 10% |
| 金龙鱼 | 100 | ⭐⭐⭐⭐ | 100 | 500-700 | 4% |
| 神秘鱼 | 200 | ⭐⭐⭐⭐⭐ | 120 | 600-800 | 1% |

#### 稀有度权重
- 稀有度越高，出现概率越低
- 特殊事件可以改变概率分布
- 鱼饵等级可以提升稀有鱼出现率

---

## 🔥 连击系统

### 机制说明
- **连击条件**：在 3 秒内连续捕获鱼
- **连击中断**：超过 3 秒未捕获鱼，连击清零
- **最大连击**：20 连击

### 连击奖励倍率
| 连击数 | 倍率 | 奖励 |
|--------|------|------|
| 1-4 | x1.0 | 无 |
| 5-9 | x1.1 | +10% |
| 10-14 | x1.2 | +20% |
| 15-19 | x1.3 | +30% |
| 20+ | x1.5 | +50% |

### 示例
```
基础分数：100 分
当前连击：12 连击
倍率：x1.2
最终得分：100 × 1.2 = 120 分
```

### 连击成就
- 🏆 连击新手：达成 5 连击（奖励 150 金币）
- 🏆 连击高手：达成 10 连击（奖励 300 金币）

## 🎉 特殊事件系统

### 事件类型

#### 1. 黄金狂潮 (Golden Rush)
- **效果**：金龙鱼和神秘鱼出现率提升
- **持续时间**：15 秒
- **概率权重**：25%
- **提示**：🌟 黄金狂潮！金龙鱼出现率提升！

#### 2. 时间奖励 (Time Bonus)
- **效果**：额外增加 10 秒游戏时间
- **持续时间**：立即生效
- **概率权重**：25%
- **提示**：⏰ 时间奖励！额外增加 10 秒！

#### 3. 双倍分数 (Double Score)
- **效果**：所有得分翻倍
- **持续时间**：15 秒
- **概率权重**：25%
- **提示**：💎 双倍分数！所有得分翻倍！

#### 4. 深海挑战 (Deep Sea)
- **效果**：只出现深海鱼（鲤鱼王、金龙鱼、神秘鱼）
- **持续时间**：15 秒
- **概率权重**：15%
- **提示**：🌊 深海挑战！只出现深海鱼！

#### 5. 宝藏出现 (Treasure)
- **效果**：神秘鱼出现率提升 5 倍
- **持续时间**：15 秒
- **概率权重**：10%
- **提示**：💰 宝藏出现！寻找特殊宝藏鱼！

### 事件机制
- **触发间隔**：30-60 秒随机
- **持续时间**：15 秒
- **事件叠加**：不叠加，新事件会覆盖旧事件
- **策略**：在双倍分数事件中尽量多捕获高分鱼

## 📋 每日任务系统

### 任务池（10 个任务）
| 任务 | 目标 | 奖励 |
|------|------|------|
| 新手钓鱼 | 捕获 5 条鱼 | 100 金币 |
| 钓鱼爱好者 | 捕获 15 条鱼 | 200 金币 |
| 高分挑战 | 单局 300 分 | 150 金币 |
| 钓鱼高手 | 单局 600 分 | 300 金币 |
| 稀有收获 | 捕获 1 条稀有鱼 | 250 金币 |
| 宝藏猎人 | 捕获 3 条稀有鱼 | 500 金币 |
| 连击高手 | 达成 5 连击 | 200 金币 |
| 连击大师 | 达成 10 连击 | 400 金币 |
| 完美一局 | 单局捕获 20 条鱼 | 600 金币 |
| 装备升级 | 升级任意装备 1 次 | 150 金币 |

### 机制说明
- **每日刷新**：每天自动刷新 3 个随机任务
- **进度追踪**：实时追踪任务进度
- **奖励领取**：完成后手动领取奖励
- **跨天重置**：未完成任务不会累积

### 策略建议
- 优先完成高奖励任务
- 稀有鱼任务和连击任务可以同时进行
- 完美一局任务需要平衡速度和数量

## 🛒 装备系统

### 3 种装备

#### 1. 鱼竿 (Rod)
- **效果**：提升鱼钩下放和回收速度
- **满级**：10 级
- **价格**：50 × 当前等级
- **策略**：优先升级，提高钓鱼效率

#### 2. 鱼线 (Line)
- **效果**：增加最大钓鱼深度
- **满级**：10 级
- **价格**：50 × 当前等级
- **策略**：中期升级，解锁深海鱼

#### 3. 鱼饵 (Bait)
- **效果**：提升稀有鱼出现概率 + 得分加成（10%/级）
- **满级**：10 级
- **价格**：50 × 当前等级
- **策略**：优先升级，直接影响分数

### 升级建议
1. **初期**（0-100 分）：优先升级鱼饵
2. **中期**（100-500 分）：平衡升级鱼竿和鱼饵
3. **后期**（500+ 分）：全部升满，追求完美

## 📊 计分公式

### 基础计分
```
基础分 = 鱼的基础分值
```

### 完整计分公式
```
最终得分 = (基础分 × 连击倍率 × 特殊事件倍率) × (1 + 鱼饵等级 × 0.1)
```

### 计算示例
```
鱼：金龙鱼（100 分）
连击：12 连击（x1.2）
特殊事件：双倍分数（x2）
鱼饵等级：5 级（+50%）

计算：
100 × 1.2 × 2 × 1.5 = 360 分
```

## 🎯 高分策略

### 1. 连击优先
- 保持连击不断
- 连击倍率是提分关键
- 5 连击以上开始有明显收益

### 2. 把握特殊事件
- 双倍分数事件中全力输出
- 黄金狂潮事件中专注深海
- 时间奖励可以延长输出时间

### 3. 装备升级顺序
```
鱼饵（得分加成）> 鱼竿（效率）> 鱼线（深度）
```

### 4. 鱼类选择
- 优先捕获高分鱼（金龙鱼、神秘鱼）
- 连击期间不要挑剔，保持连击
- 特殊事件期间针对性钓鱼

### 5. 每日任务
- 每天完成任务获取额外金币
- 完美一局任务奖励最高
- 连击任务可以和分数任务同时进行

## 🔄 游戏循环

### 单局流程
```
开始游戏 → 下放鱼钩 → 捕获鱼 → 获得分数 → 完成任务
→ 触发特殊事件 → 保持连击 → 时间结束 → 提交分数
```

### 长期循环
```
玩游戏 → 获得金币 → 升级装备 → 解锁成就 → 完成任务
→ 挑战高分 → 冲击排行榜 → 分享炫耀
```

## 📈 数值平衡

### 预期进度
- **新手期**（0-100 分）：1-2 局
- **成长期**（100-500 分）：3-5 局
- **成熟期**（500-1000 分）：5-10 局
- **高手期**（1000+ 分）：10+ 局

### 装备收益
- 满级鱼饵：+100% 得分
- 满级鱼竿：+50% 速度
- 满级鱼线：+100% 深度

### 成就难度
- 简单（50% 玩家可达）：初次钓鱼、钓鱼新手
- 中等（20% 玩家可达）：钓鱼达人、小有所成
- 困难（5% 玩家可达）：钓鱼大师、高分高手、装备大师
