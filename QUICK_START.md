# 快速开始指南

## 🚀 5 分钟上手

### 前置条件
- Cocos Creator 3.8 或更高版本
- 微信开发者工具（用于微信小游戏发布）

### 步骤 1：打开项目
```bash
# 在 Cocos Creator 中打开项目
# 文件 → 打开项目 → 选择 fishing-wechat 目录
```

### 步骤 2：搭建场景
按照 `SCENE_SETUP_GUIDE.md` 文档搭建主场景：
1. 创建 Canvas 和背景
2. 创建 GameManager 节点并添加所有脚本
3. 创建 UI 界面（主菜单、游戏界面、商店等）
4. 创建鱼钩和鱼类预制体
5. 绑定所有引用

**提示**：如果不想手动搭建，可以：
- 使用 Cocos Creator 的资源商店模板
- 或等待后续提供的完整场景文件

### 步骤 3：配置微信小游戏（可选）
如果只在编辑器测试，可以跳过此步骤。

如果要发布到微信：
1. 获取微信 AppID（https://mp.weixin.qq.com）
2. 打开 `wechat-integration.md`
3. 按照指南配置项目

### 步骤 4：运行测试
1. 在 Cocos Creator 中点击 ▶️ 运行按钮
2. 点击"开始钓鱼"按钮
3. 点击屏幕任意位置下放鱼钩
4. 捕获鱼类获得分数
5. 时间结束后查看最终得分

### 步骤 5：构建发布
```bash
# 在 Cocos Creator 中
# 菜单 → 项目 → 构建发布
# 选择平台：WeChat Game
# 点击构建
```

## 🎮 游戏操作

### 基本操作
- **点击屏幕**：下放鱼钩
- **鱼钩自动回收**：到达最大深度或捕获鱼后
- **目标**：在 60 秒内获得最高分

### 鱼类分值
| 鱼类 | 分值 | 稀有度 | 出现深度 |
|------|------|--------|----------|
| 小鲤鱼 | 10 | ⭐ | 浅 |
| 鲫鱼 | 20 | ⭐⭐ | 中浅 |
| 草鱼 | 30 | ⭐⭐ | 中 |
| 鲤鱼王 | 50 | ⭐⭐⭐ | 中深 |
| 金龙鱼 | 100 | ⭐⭐⭐⭐ | 深 |
| 神秘鱼 | 200 | ⭐⭐⭐⭐⭐ | 最深 |

### 装备系统
在商店可以购买升级：
- **鱼竿**：提升鱼钩速度
- **鱼线**：增加最大深度
- **鱼饵**：提高稀有鱼出现概率 + 得分加成

## 📦 资源替换

### 临时资源
当前项目使用色块作为临时资源：
- 鱼类：不同颜色的矩形
- 鱼钩：灰色圆形
- 背景：纯色

### 替换正式资源
1. 准备图片资源（PNG 格式）
2. 导入到 `assets/textures/` 目录
3. 在场景中替换 Sprite 的 SpriteFrame
4. 调整大小和锚点

### 推荐资源尺寸
| 资源 | 推荐尺寸 | 格式 |
|------|---------|------|
| 鱼类 | 80-120 x 50-70 | PNG |
| 鱼钩 | 40 x 40 | PNG |
| 背景 | 720 x 1280 | JPG |
| 按钮 | 200-300 x 60-80 | PNG |
| UI 图标 | 64 x 64 | PNG |

## 🐛 常见问题

### Q: 点击没反应？
A: 检查：
1. HookController 是否正确添加到 GameManager 节点
2. 碰撞体是否启用
3. 控制台是否有错误信息

### Q: 鱼不生成？
A: 检查：
1. FishManager 的 fishPrefabs 是否绑定了预制体
2. spawnInterval 是否设置合理（默认 2 秒）
3. maxFishCount 是否太小

### Q: 分数不增加？
A: 检查：
1. 鱼的碰撞体是否正确
2. FishData 组件是否添加到鱼预制体
3. onFishCaught 回调是否正确绑定

### Q: 微信登录失败？
A: 检查：
1. 是否在微信环境运行
2. AppID 是否正确配置
3. 后端服务器是否正常

### Q: 构建后无法运行？
A: 检查：
1. 微信开发者工具版本
2. AppID 配置
3. 构建参数是否正确
